Información del Tema

La recursividad es una técnica poderosa que permite resolver problemas complejos mediante la división en subproblemas similares. Esta técnica requiere dos componentes fundamentales: una condición base que detenga la recursión, y una llamada recursiva que acerque el problema a dicha condición base. Si cualquiera de estos elementos falta, el programa podría caer en un bucle infinito y ocasionar un desbordamiento de pila (stack overflow).

Además de los ejemplos clásicos como el cálculo del factorial y la sucesión de Fibonacci, la recursividad se aplica en algoritmos de búsqueda y ordenamiento. Por ejemplo, en el recorrido de árboles binarios (preorden, inorden y postorden) y en algoritmos como Quicksort y MergeSort.

Es importante destacar las desventajas de la recursividad: un mayor consumo de memoria debido al mantenimiento de múltiples llamadas activas en la pila, y un posible impacto en el rendimiento. Sin embargo, cuando se implementa correctamente, puede simplificar significativamente el diseño de algoritmos complejos, haciéndolos más legibles y modulares.

Programa

public class Recursividad {

    // Método recursivo para calcular el factorial
    public static int factorial(int n) {
        if (n == 0) {  // Caso base
            return 1;
        } else {
            return n * factorial(n - 1);  // Llamada recursiva
        }
    }

    public static void main(String[] args) {
        int numero = 5;
        System.out.println("El factorial de " + numero + " es: " + factorial(numero));
    }
}
  public class Recursividad {

    // Método recursivo para calcular el n-ésimo número de Fibonacci
    public static int fibonacci(int n) {
        if (n <= 1) {  // Caso base
            return n;
        } else {
            return fibonacci(n - 1) + fibonacci(n - 2);  // Llamada recursiva
        }
    }

    public static void main(String[] args) {
        int numero = 6;
        System.out.println("El número de Fibonacci en la posición " + numero + " es: " + fibonacci(numero));
    }
}
class Nodo {
    int valor;
    Nodo izquierdo, derecho;

    // Constructor
    public Nodo(int valor) {
        this.valor = valor;
        izquierdo = derecho = null;
    }
}

public class ArbolBinario {

    Nodo raiz;

    // Método recursivo para realizar la búsqueda en el árbol
    public boolean buscar(Nodo raiz, int valor) {
        // Caso base: Si el árbol está vacío o hemos encontrado el valor
        if (raiz == null || raiz.valor == valor) {
            return raiz != null;
        }

        // Si el valor es menor que el valor de la raíz, buscar en el subárbol izquierdo
        if (valor < raiz.valor) {
            return buscar(raiz.izquierdo, valor);
        }

        // Si el valor es mayor, buscar en el subárbol derecho
        return buscar(raiz.derecho, valor);
    }

    public static void main(String[] args) {
        ArbolBinario arbol = new ArbolBinario();
        arbol.raiz = new Nodo(10);
        arbol.raiz.izquierdo = new Nodo(5);
        arbol.raiz.derecho = new Nodo(15);
        arbol.raiz.izquierdo.izquierdo = new Nodo(3);
        arbol.raiz.izquierdo.derecho = new Nodo(7);

        int valorBuscar = 7;
        System.out.println("¿El valor " + valorBuscar + " está en el árbol? " + arbol.buscar(arbol.raiz, valorBuscar));
    }
}
